> [!info] Задача
> Создать базовый транслятор ВМ, предназначенный для реализации арифметико- логических и push/pop-команд языка ВМ. Для этой версии транслятора ВМ предполагается, что исходный код ВМ не содержит ошибок. Проверку и обработку ошибок можно будет добавить в последующие версии транслятора ВМ, но в проект 7 они не входят.
